Fantastic Frank

"Fantastic Frank" Comic Book Cover"

Illustrating and designing the cover for an issue of a children's action comic book.

  • Project Type: Assignment
  • Completed: April 2019
  • Tools used: Adobe Illustrator

Creative Strategy: The goal of this project was to illustrate a children's super hero comic book cover using various Adobe Illustrator techniques. We were given a lot of creative freedom in choosing the art direction of our illustration, so I decided to use this opportunity to challenge myself. I have done a few projects in the past using a geometric, flat design style illustration, so for this project I wanted to attempt a modern retro style as inspired by older Steamboat Willie cartoons. This allowed me to challenge myself with using different Illustrator tools and techniques in order to achieve the effect I wanted. This project also allowed me to practice techniques such as text effects in order to create the "pop-out" effect on the main title.
